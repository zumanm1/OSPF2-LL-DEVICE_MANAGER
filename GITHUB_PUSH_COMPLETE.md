## üìã How to Clone and Use (For New Users)

### 1. Clone the Repository

```bash
# Via HTTPS (Recommended)
git clone https://github.com/zumanm1/OSPF2-LL-DEVICE_MANAGER.git
cd OSPF2-LL-DEVICE_MANAGER

# Via SSH
git clone git@github.com:zumanm1/OSPF2-LL-DEVICE_MANAGER.git
cd OSPF2-LL-DEVICE_MANAGER

# Via GitHub CLI
gh repo clone zumanm1/OSPF2-LL-DEVICE_MANAGER
cd OSPF2-LL-DEVICE_MANAGER
```

### 2. Install Dependencies

```bash
# Backend
cd backend
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
cd ..

# Frontend
cd frontend
npm install
cd ..
```

### 3. Start the Application

```bash
# Terminal 1 - Backend
cd backend
source venv/bin/activate
python3 -m uvicorn server:app --host 0.0.0.0 --port 9050 --reload

# Terminal 2 - Frontend
cd frontend
npm run dev
```

### 4. Access the Application

```
Frontend: http://localhost:5173
Backend API: http://localhost:9050/docs
Login: admin / admin123
```
